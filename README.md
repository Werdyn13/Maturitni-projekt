# Mobilní aplikace pro pekárnu
Mobilní aplikace pro pekárnu, umožní rychlé objednání zboží a kontrolu vyzvednutí zboží. Zaměstnanci se můžou podívat na denní výrobu, směny a nástěnku, pomocí které můžou komunikovat s vedoucím. Vedoucí může potvzovat nové profily zákazníků, řešit dotazy zaměstnanců a přijímat objednávky.
