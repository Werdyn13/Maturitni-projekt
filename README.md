# Mobilní aplikace pro pekárnu
Mobilní aplikace pro pekárnu, umožní rychlé objednání zboží a kontrolu vyzvednutí zboží. Zaměstnanci se můžou podívat na denní výrobu, směny a nástěnku, pomocí které můžou komunikovat s vedoucím. Vedoucí může potvzovat nové profily zákazníků, řešit dotazy zaměstnanců a přijímat objednávky.

## Prezentace: přidání věcí (do další hodiny)
- obrázky k tomu co je hotové, např. admin page atd.
- okomentovat jiné programy, v čem můžou být lepší, neříkat že sem je nezkoušel nebo se nelíbili
- při prezentování neříkat když nějaké malé věci nefungují
- pokud něco většího nejde, vymyslet si důvod(projev) proč to nejde
- stránku s nástroji dat na jeden slide a krátký popis ke každé z nich
- ukázka něčeho, co bylo složitější, zajímavější, těžší než jsem čekal
- více konkrétní prezentace, ne tak moc obecná, ukázat obrázky(příklady) toho jak projekt vypadá
- natočit si video finální verze projektu, kdyby se náhodou stalo, že by projekt nešel spustit(může se stát cokoliv)
- netrávit moc času na technologiích
- VYMĚNIT TECHNOLOGIE ZA KNIHOVNY
- přidat odkazy na chaty
